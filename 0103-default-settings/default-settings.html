Formula to convert pixels to ems is
  (target px) divided by (context px) = (value em)
  where 'context px' is parent value (in most cases font size of html)


All browser agreed upon default behavior when html font-size is 100% and line-height is 1.
=====    ======================     =========================   =====================    ===========================
         pixels                     em                          pixels to em formula     pixels to em formula
tag      font-size   line-height    font-size     line-height   font-size                line-height
=====    =======================    =========================   ======================   ============================
html     16px        16px           1em            1em          16px / 16px = 1em        16px / 16px = 1em 
body     16px        16px           1em            1em          16px / 16px = 1em        16px / 16px = 1em 
p        16px        16px           1em            1em          16px / 16px = 1em        16px / 16px = 1em 
ul       16px        16px           1em            1em          16px / 16px = 1em        16px / 16px = 1em 
h1       32px        32px           2em            2em          32px / 16px = 2em        32px / 16px = 2em
h2       24px        24px           1.5em          1.5em        24px / 16px = 1.5        24px / 16px = 1.5


Theses values are not the same across browsers. Chrome values are listed below.
